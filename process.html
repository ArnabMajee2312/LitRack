PROCESS WRITING

When something is made or done, we have to write about the process of making or doing that thing. This is called Process.

1. At first, we have to add a Title.

Example:

HOW TEA IS PREPARED/TEA PREPARATION/THE MAKING OF TEA

2. Secondly, we have to write about the advantages of the product.

Example:

Tea is a popular drink which makes us refresh.

3. We have to write every sentence in Common Tense.

Example:

Example: It’s prepared/proposed/made/completed through some stages/steps.

4. Next, we have to start writing by this word “At first”/ “First”.

5. Then, we have to write sentences by this format: Linker – Object – is/are – 3rd form of Verb – Other.

Example:

At first, Water is taken in a pot.

6. Some linkers have to add with every sentence. These are: “Then, Next, Later, After that, Thereafter, Now, etc.”

*We can’t use same linker in case of consecutive sentences.

7. We have to write “Finally” at the last sentence of the process.

Example:

Finally Tea is ready to be served.
